# How to use
typescript
```js
import { DateUtils } from "ts-commons";

DateUtils.toString(new Date(), "yyyy-MM-dd HH:mm:ss.SSS");
```

javascript
```js
var tsCommons = require("ts-commons");

tsCommons.DateUtils.toString(new Date(), "yyyy-MM-dd HH:mm:ss.SSS");
```
